# 🐔 CluckBook

A modern cloud-powered chicken coop management application built with **Next.js, React, TypeScript, Tailwind CSS, Firebase, Firestore, and Vercel**.

Track your flock, monitor egg production, manage health records, record expenses, and organize daily farm tasks from one responsive dashboard.

## ✨ Features

### 🐔 Flock Management
- Favorite hens
- Breed tracking
- Hatch date
- Automatic age calculation
- Personal notes

### 🥚 Egg Production
- Daily egg tracking
- Weekly totals
- Dashboard statistics

### 🌾 Daily Care
- Feed logs
- Cleaning logs
- Weight tracking

### 🩺 Health Records
- Illness tracking
- Medications
- Vaccinations
- Vet visits
- Recovery notes

### 💰 Expense Tracking
- Feed expenses
- Bedding
- Medication
- Vet bills
- Equipment
- Monthly spending summary

### ✅ Farm Tasks
- Daily checklist
- Task completion tracking
- Remaining task count
- Dashboard summary

### ☁️ Cloud Sync
- Firebase Firestore
- Cross-browser synchronization
- Cross-device synchronization
- Automatic cloud persistence
